# Deployment Guide

## Quick Start Options

### 1. Cloud Deployment (Recommended)
1. Visit workshop-manager.com/deploy
2. Choose cloud provider
3. Enter environment variables
4. Click Deploy

### 2. Self-Hosted Deployment

#### Prerequisites
- Server with 4GB RAM minimum
- Domain name
- SSL certificate
- PostgreSQL database
- Node.js 18+

#### Steps
1. Clone repository
2. Configure environment
3. Build application
4. Start server
